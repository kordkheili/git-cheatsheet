| Commands                             | Description |
| ------------------------------------ | ----------- |
| git status                           |             |
| git add [file.txt]                   |             |
| git add -A                           |             |
| git reset [file.txt]                 |             |
| git clean -fd                        |             |
| git diff                             |             |
| git commit -m "your message here..." |             |

| -------------------------------------- | ----------- |
